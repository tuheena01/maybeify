## 🚀 Quick Start

### Prerequisites
- Node.js (v16 or higher)
- Yarn package manager

### Installation

1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/maybeify-website.git
cd maybeify-website
```

2. Install dependencies
```bash
cd frontend
yarn install
```

3. Start development server
```bash
yarn start
```

The website will open at `http://localhost:3000`

4. Build for production
```bash
yarn build
```

## 🎨 Customization

### Update Content
Edit `/frontend/src/data/mock.js` to change:
- Books, authors, testimonials
- Pricing packages
- Contest information
- Social media links

### Change Colors
Edit `/frontend/src/index.css`:
```css
--primary: 221 83% 22%;   /* Navy Blue */
--accent: 43 89% 58%;     /* Gold */
```
